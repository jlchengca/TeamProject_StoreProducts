I have learned many things through finishing this Full Stack Application Team Project, especially in the following aspects:

1. Database connection issues:
I learned that an incorrect MongoDB URI (e.g., wrong password or missing database name) prevents the API from returning data.
Evidence: My five-products endpoint failed until I corrected the DB_URI in Replit Secrets.

2. Environment configuration:
I realized that local code working in VS Code does not guarantee it will work in deployment environments like Replit.
Evidence: My API worked on localhost:3000 but failed online until environment variables were properly set.

3. API testing and debugging skills
I improved my ability to test endpoints using browser and logs.
Evidence: I used console logs and checked response status to debug why five-products did not return data.
